# 🪴 ZEN GARDEN

i made a little browser game where you can mow grass and plant flowers. no frameworks. no libraries. just vibes and vanilla javascript

## what you can do
* mow grass by clicking and dragging
* plant roses, daisies, tulips, sunflowers, mushrooms, stones
* add ponds that merge together
* release butterflies that float around gently + birds that soar across the sky
* watch it transition from day to night with stars and a moon
* listen to original 8-bit music made entirely with the web audio api (night jazz and sakura waltz)
* randomize the whole garden if you want chaos
* save your garden as a file and come back to it later

## the tech stuff
pure html canvas + web audio api. zero dependencies. one file.
